# AIM_Trainer_pas

AIM Trainer на PascalABC.NET + GraphABC.

## Возможности

- Главное меню с выбором режима.
- Classic mode: цель поражается левой кнопкой мыши.
- Dual mode: голубая цель поражается ЛКМ, красная цель поражается ПКМ.
- Обратный отсчёт перед стартом.
- HUD со счётом, FPS, точностью, средним временем реакции, оставшимися жизнями и временем.
- Экран результатов после завершения сессии.
- График последних 50 реакций.
- Корректная обработка одиночного клика за кадр без двойного подсчёта.

## Управление

- ЛКМ — стрельба по голубой цели.
- ПКМ — стрельба по красной цели в Dual mode.
- `Space` или `Esc` во время игры — вернуться в меню.
- `R` на экране результатов — начать заново последний выбранный режим.
- Клик на экране результатов — вернуться в меню.

## Правила

Игра длится 60 секунд или до 15 промахов. Любой клик считается выстрелом. Попадание засчитывается только если нажата правильная кнопка и курсор находится внутри текущей цели. Неверная кнопка дополнительно учитывается в счётчике `Wrong buttons`.

## Запуск

Откройте `AIM_Trainer.pas` в PascalABC.NET и запустите программу. Для работы требуется стандартный модуль `GraphABC`.
